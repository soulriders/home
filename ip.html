5대 기술 해자는 경제성과 안전성을 증명하는 근거입니다.

SDA IP는 기체를 더 단순하게 만들고, 실패 시나리오를 설명 가능한 비행 구조로 바꾸는 기술 자산입니다.

기술로 돌아가기

IP를 보는 기준

Receivepower의 IP는 기술명 나열보다 구조 변화와 운항 효과로 읽어야 합니다. 각 기술 해자는 `구성 → 변화 → 효과`의 순서로 경제성과 안전성에 연결됩니다.

  • 경제성기계식 부품, 생산 공정, 정비 항목을 줄이는 방향
  • 안전성전환, 고장, 동력 상실, 상태 인식을 설명 가능한 구조로 만드는 방향
등록

1. 독립제어 틸팅 추력기관

각 추력기관을 같은 각도로 한 번에 움직이는 방식에서 벗어나, 전환 구간에서 추력과 양력을 나누어 다루는 구조입니다.

  • 구성추력기관별 독립 틸팅과 분산 추진 제어
  • 변화수직비행에서 고정익 순항으로 넘어가는 전환 구간을 세분화
  • 효과VTOL 전환 안정성, 유인비행 확장성
출원

2. 가상 조종면 기반 비행 제어

물리 조종면, 힌지, 링크, 액추에이터 의존을 줄이고 추진기 배치와 추력 차이를 비행 제어 자원으로 사용하는 구조입니다.

  • 구성분산 추진기의 위치와 위상 차이를 제어 자원으로 사용
  • 변화가동 조종면과 관련 기계 부품 축소
  • 효과기체 원가, 정비 항목, 생산 복잡도 감소
출원

3. 이산 매트릭스 기반 비행 관리 시스템

비행 상태와 고장 상태를 사전에 나누고, 각 상태에 맞는 제어 모드를 전환하는 결정론적 비행 관리 구조입니다.

  • 구성상태별 제어 매트릭스와 고장 대응 모드
  • 변화블랙박스식 판단보다 시험 가능한 제어 단위로 분리
  • 효과인증 설명력, 운항 로그, 실패 대응 구조
출원

4. 무동력 고양력 제어

동력 상실 또는 저속 구간에서 기체가 가진 양력과 회전 요소를 활용해 착륙 여유와 방향 제어 가능성을 확보하는 독립 발명입니다.

  • 구성무동력 상태의 양력, 항력, 회전 거동 활용
  • 변화비상 상황에서 추가 시간과 공간을 확보
  • 효과동력 상실 시나리오, 비상 착륙 안전성
출원

5. SADS 기반 가상피토관

단일 물리 센서에만 의존하지 않고 여러 신호를 융합해 기체의 공기 속 상태를 추정하는 독립 발명입니다.

  • 구성GNSS, IMU, 추진기 데이터, 센서융합 기반 상태 추정
  • 변화기체가 현재 어떤 공기 상태에 놓였는지 판단하는 보조 채널 확보
  • 효과상태 인식, 모드 전환, 제어 안정성

5대 기술 해자의 연결

  • 트럭 수준의 경제성가상 조종면, 무조종면 구조, 생산·정비 단순화
  • 유인비행의 안전성틸팅 전환, 이산 제어, 무동력 고양력, SADS 상태 추정
  • SDA운항 조건과 실패 시나리오에서 출발하는 비행체 아키텍처
